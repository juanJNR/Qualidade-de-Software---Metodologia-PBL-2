## 🎯 Objetivo do Repositório
Este repositório reúne as atividades desenvolvidas na disciplina de
Qualidade de Software, realizadas por meio da metodologia PBL, referentes
ao sistema **LocalEats**.

## 📚 Organização
- `docs/` → Atividades realizadas ao longo das aulas
- `artefatos/` → Diagramas, relatórios e evidências
- `src/` → Código-fonte (quando aplicável)
- `referencias/` → Materiais de apoio e bibliografia
